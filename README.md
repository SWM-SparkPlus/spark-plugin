# SparkPlus
Spark+는 H3, 위/경도 좌표 등의 공간 정보를 국내 주소체계(신주소/구주소)와 함께 처리할 수 있도록 지원하는 Plugin입니다.

Spark+는 Apache Spark와 MySQL을 지원합니다.

## Setup
---
Spark+는 Pypi에 배포되어 있으며, 다음 커맨드로 설치할 수 있습니다.
```
pip install sparkplus
```
